# Pim pam pum

Hay que ir contando de 1 a ... ; si toca 3 o múltiplo de tres, se dice Pim; si toca 5 o múltiplo de cinco, se dice Pam; si toca 7 o múltiplo de 7, se dice Pum.

- Pim - pasa al siguiente
- Pam - cambia sentido
- Pum - se salta al siguiente

El que falle bebe y se reanuda el juego.

Si fallas, bebes un trago; si te has acabado la cerveza, quedas eliminado; el ganador es el que más aguanta.
